# Sistema de Facturación

Sistema web empresarial para la gestión de facturación e inventario, desarrollado en ASP.NET WebForms con C# y SQL Server.

> Proyecto en desarrollo activo.

---

## Módulos

- Autenticación y control de acceso
- Gestión de clientes
- Gestión de productos e inventario
- Facturación
- Administración de países y provincias

---

## Tecnologías

- ASP.NET WebForms (.NET Framework)
- C#
- CSS3
- SQL Server

---

## Requisitos

- Visual Studio 2019 o superior
- .NET Framework 4.8
- SQL Server 2017 o superior
- SQL Server Management Studio (SSMS)

---

## Instalación

1. Clonar el repositorio
```bash
git clone https://github.com/AsTazx/SistemaDeFacturacion.git
```

2. Restaurar la base de datos ejecutando el archivo `database.sql` incluido en el repositorio desde SQL Server Management Studio.

3. Abrir `SistemaDeFacturacion.sln` en Visual Studio.

4. Actualizar el connection string en `Web.config` apuntando a tu instancia de SQL Server.

5. Compilar y ejecutar con **F5**.
